# Reflow Profile

## Temperature Specifications

| Parameter            | Specification                     |
| :------------------- | :-------------------------------- |
| **Peak Temperature** | < 250°C (Recommended 245 +0/-5°C) |
| **Preheat**          | 150–180°C for 55–65 seconds       |
| **Time above 200°C** | 65–75 seconds                     |
| **Ramp Rate (Up)**   | Max 2°C/s                         |
| **Ramp Rate (Down)** | Max 2.5°C/s                       |
| **Max Passes**       | ≤ 2 times                         |

## Reflow Phases

### Preheating

- Temperature: 150–180°C
- Duration: 55–65 seconds

### Soaking

- Temperature above 200°C
- Duration: 65–75 seconds

### Peak

- Peak Temperature: < 250°C (Recommended 245 +0/-5°C)

### Cooling

- Ramp Rate: Max 2.5°C/s

## Standards Compliance

- BLE 4.2
- IEEE 802.15.1

::: warning
Do not exceed 2 reflow passes. Peak temperatures above 250°C may cause permanent damage to the module.
:::
