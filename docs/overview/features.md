# Product Features

## Core Features

- **Ultra-small size** low-power Bluetooth 5.2 module
- **Built-in 12-bit high-precision ADC**
- **Deep sleep current:** 0.8µA

## Supported Interfaces

| Interface | Description                 |
| :-------- | :-------------------------- |
| **UART**  | Serial communication        |
| **GPIO**  | General Purpose I/O         |
| **I2C**   | Inter-Integrated Circuit    |
| **PWM**   | Pulse Width Modulation      |
| **ADC**   | Analog-to-Digital Converter |
| **SPI**   | Serial Peripheral Interface |
| **JTAG**  | Debugging interface         |

## Communication Features

- Supports command-based switching between **AT mode** and **Transparent transmission mode**
- Supports AT command configuration
- **Communication distance:** Up to 40m in open areas at 0dBm transmit power
- **Max packet size:** 20 bytes per frame

## Power Features

- **Power supply:** 1.7V to 3.3V (3.0V recommended)
- Pins can enable the serial port to switch between working and sleep states

::: tip
The module achieves ultra-low power consumption of just 0.8µA in deep sleep mode, making it ideal for battery-powered IoT applications.
:::
