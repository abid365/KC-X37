# XW37-BLE 5.2 Module

**Version:** V1.0  
**Date:** 2023.12.13  
**Manufacturer:** XW-CHIP TECHNOLOGY

## Overview

The **XW37-BLE** is a low-power Bluetooth 5.2 module. It integrates a high-performance 2.4GHz RF transceiver, a feature-rich baseband processor, a 32-bit MCU running at 32 MHz, and various peripheral interfaces.

## Key Specifications

| Specification              | Value                          |
| :------------------------- | :----------------------------- |
| **Bluetooth Version**      | 5.2                            |
| **MCU**                    | 32-bit @ 32 MHz                |
| **Flash**                  | 2 Mbit SiP                     |
| **RAM**                    | 32 KB                          |
| **ROM**                    | 32 KB                          |
| **Communication Distance** | Up to 40m (open area @ 0dBm)   |
| **Max Packet Size**        | 20 bytes/frame                 |
| **Power Supply**           | 1.7V - 3.3V (3.0V recommended) |
| **Dimensions**             | 18mm × 12mm × 1.7mm            |
