# Package Dimensions

## Module Dimensions

| Parameter           | Value      |
| :------------------ | :--------- |
| **Length**          | 18 mm      |
| **Width**           | 12 mm      |
| **Height**          | 1.7±0.2 mm |
| **Pin Spacing**     | 1.25 mm    |
| **Board Thickness** | 0.8 mm     |

## Physical Specifications

The XW37-BLE module features an ultra-compact design measuring just 18mm × 12mm × 1.7mm, making it suitable for space-constrained IoT applications.

::: info
The compact 18mm × 12mm footprint makes the XW37-BLE ideal for small form-factor devices and wearable applications.
